# Screenshots Directory

This directory contains screenshots demonstrating the E-Commerce Store application features.

## Required Screenshots

Please add the following screenshots to this folder:

1. `homepage.png` - Homepage with category grid and featured products
2. `category-page.png` - Category page showing products in a specific category
3. `cart-page.png` - Shopping cart with items and coupon application
4. `checkout-success.png` - Purchase success confirmation page
5. `admin-dashboard-products.png` - Admin dashboard showing products management
6. `admin-dashboard-analytics.png` - Admin dashboard showing analytics and charts

## File Naming Convention

- Use lowercase letters
- Use hyphens to separate words
- Use `.png` or `.jpg` format
- Keep file names descriptive and consistent
